# 開発進捗

## 概要
本ドキュメントは、就労継続支援A型事業所向け「期限・進捗管理システム」の開発における全体的な進捗状況、発生した問題、決定事項、および次のステップを継続的に記録するためのものです。
`/onboard` や `/sync` 実行時にAIエージェントが参照・追記を行うことで、開発の文脈を常に最新に保ちます。

* **直近の課題**: なし
