# 「A型事業所基本セット」 デフォルト期限項目一覧

※以下の「計算パターンの設定値」はシステム導入時の初期値であり、納品後に管理者様が設定画面から自由に変更できる前提とします。

## 【1】 受給者証・行政手続き関連

他機関（役所や病院）が絡むため、リードタイムを長めに設定し、3段階のアラートで確実に回収するグループです。

**① 支給決定期間**

* **計算パターンの例:** 【月末丸め】開始日から1年後の月末
* **アラート設定:** 3段階（6週間前[黄] ＞ 4週間前[橙] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `案内済み（提出待ち）` ＞ `完了（更新）`

**② 利用者負担の適用期間**

* **計算パターンの例:** 【月末丸め】開始日から1年後の月末
* **アラート設定:** 3段階（6週間前[黄] ＞ 4週間前[橙] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `案内済み（提出待ち）` ＞ `完了（更新）`

**③ 在宅利用期間（対象者のみ）**

* **計算パターンの例:** 【加算】開始日から〇ヶ月後（※自治体や個別の決定に準ずる）
* **アラート設定:** 3段階（6週間前[黄] ＞ 4週間前[橙] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `案内済み（提出待ち）` ＞ `完了（更新）`

**④ 障害者手帳の有効期限 / 自立支援医療受給者証（※必要な場合）**

* **計算パターンの例:** 【ルールなし】（手帳の種類等によりバラバラなため、初回のみ手動入力し、次回からは必要に応じて手動更新）
* **アラート設定:** 3段階（6週間前[黄] ＞ 4週間前[橙] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `案内済み（提出待ち）` ＞ `完了（更新）`

## 【2】 支援計画・モニタリング関連

事業所内部での書類作成と、利用者からの「押印・署名」が必須となるグループです。実地指導対策として、フローを細かく管理します。

**⑤ 個別支援計画（本作成・見直し）**

* **計算パターンの例:** 【反復】利用開始日（または前回作成日）から6ヶ月ごと
* **アラート設定:** 2段階（4週間前[黄] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `面談済み` ＞ `書類作成済み` ＞ `署名・押印済み（完了・更新）`

**⑥ モニタリング**

* **計算パターンの例:** 【反復】利用開始日（または前回作成日）から6ヶ月ごと
* **アラート設定:** 2段階（4週間前[黄] ＞ 2週間前[赤]）
* **ステータスフロー:** `未対応` ＞ `面談済み` ＞ `書類作成済み` ＞ `署名・押印済み（完了・更新）`
