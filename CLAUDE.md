# AIアシスタント 開発ルール (Custom Instructions)

今後開発を進める上で、AIは以下の指示を**絶対に**遵守すること。

## 1. 開発前の参照
* **「コードを書く前に必ず `docs/` 配下のドキュメントを確認すること」**
* 要件定義、アーキテクチャ、スキーマ、UI/UX仕様、ロードマップ、開発進捗と齟齬がないように実装方針を決定する。

## 2. 完了時の更新
* **「タスクが終了したら、進捗管理ファイル（ロードマップ等）にチェックを入れること」**
* `/sync` コマンドの実行や手動での更新を通じて、常に `docs/04_development_roadmap.md` および `docs/開発進捗.md` を最新の状態に保つ。
